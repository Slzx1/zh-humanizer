# zh-humanizer

中文 AI 写作降痕 Skill —— 让 AI 生成的文字读起来像人写的。

基于 [humanizer](https://github.com/blader/humanizer) 的英文架构，结合中文语言学和 AIGC 检测学术研究（ACL 2025/CCL 2025/
《语言文字应用》2025 等 60+ 来源），全面重构为适配中文语境的检测与改写系统。

## 安装

```bash
mkdir -p ~/.claude/skills
cp -r zh-humanizer ~/.claude/skills/zh-humanizer
```

## 使用

```
/zh-humanizer

[粘贴需要去 AI 味的中文文本]
```

或直接对话：

```
帮我去掉这段文字的 AI 味：[文本]
```

### 声音校准

提供你自己的写作样本，让改写匹配你的个人风格：

```
/zh-humanizer

这是我之前写的文字，用来匹配风格：
[粘贴 2-3 段你自己的写作]

现在帮我去 AI 味：
[粘贴需要改写的 AI 文本]
```

Skill 会分析你的句子节奏、用词习惯和标点偏好，在改写中复现这些特征。

## 42 个检测模式

### 内容模式 (7)
1. 意义膨胀：强行拔高、上价值
2. 强行点名权威媒体/名人
3. 伪深刻：万能"-ing"式分析
4. 宣传腔/广告语
5. 模糊引用/虚张声势
6. 模板化「挑战与展望」段落
7. 强行三段式排比

### 语言与语法模式 (9)
8. AI 高频词汇黑名单
9. 「进行+名词」过度名词化
10. 「是」的逃避症
11. 「不是……而是……」句式密度过高
12. 「随着……的发展」万能开头
13. 「首先……其次……最后……」机械结构
14. 「一方面……另一方面……」平衡论述
15. 被动句密度过高（英语口音）
16. 抽象名词做主语（无灵主语句）

### 风格模式 (9)
17. 破折号频率异常
18. Markdown 格式残留
19. 标题过度格式化
20. Emoji 装饰
21. 引号格式错误
22. 段末总结句
23. 段落长度高度均匀
24. 「值得关注的是」及变体
25. 段末互动钩子

### 沟通模式 (3)
26. Chatbot 口头禅
27. 甜腻迎合语气
28. 知识截止日期声明 + 信息空白填充

### 填充与模糊模式 (4)
29. 填充词替换表
30. 过度模糊限制
31. 笼统的正面结论
32. 「可以预见的是……」展望式收尾

### 中文特有模式 (8)
33. 四字格轰炸
34. 成语错用/堆砌/过度
35. 「的」字密度异常
36. 语气词缺失
37. 翻译腔：中文说英文的话
38. 「最」字高频（豆包体）
39. AI 小说/创意写作专项
40. 诗词/名言强行引用

### 附加指导
- 什么不算 AI 痕迹（避免误判）
- 人类写作的信号（保留这些）
- 工作流：草稿 → 审计 → 终稿

## 核心洞察
